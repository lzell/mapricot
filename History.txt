== 0.0.2 2009-03-17

Nokogiri and Libxml support.

Select the parser to use by setting Mapricot.parser as follows:

Mapricot.parser = :nokogiri
Mapricot.parser = :libxml
Mapricot.parser = :hpricot

Hpricot will be used by default
